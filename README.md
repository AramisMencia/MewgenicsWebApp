# Mewgenics Web App

Herramienta web para jugadores de Mewgenics que permite gestionar gatos, analizar cruces y visualizar árboles genealógicos.

---

## Qué hace

Permite registrar tus gatos y entender mejor sus relaciones y posibles cruces, facilitando decisiones de breeding fuera del juego.

---

## Funcionalidades

### Gestión de gatos
Crear, editar y eliminar gatos con:
- Nombre
- Stats  
- Género
- Orientacion
- Color  
- Padres (Opcional)

### Árbol genealógico
Visualización interactiva en forma de grafo:
- Zoom  
- Drag  

### Matchmaking
Evaluación de cruces con filtros:

-Sistema de scoring basado en:
 * Mejora de estadísticas
 * Diversidad genética
 * Compatibilidad

-Filtros:
* Evitar mismo género
* Evitar parentesco (configurable)
* Priorizar atributos específicos según objetivo del jugador

### Import / Export
- Guardar datos en JSON  
- Cargar datos existentes  

### Sistema de mundos
- Cada usuario maneja su propio datos
- No requiere login
